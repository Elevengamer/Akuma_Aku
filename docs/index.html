Styling

For styling you can add a "style" tag or you import a css file

    
        <style>
            /* CSS-Code goes here */
        </style>
Video for Details

You have multiple options to style your website using css. You can add a class="" and an id="" to your tag for even more options

        
            <div class="classNameHere" id="idHere">Just a noraml div</div>

Some css code to style that

        
            /* File Start */

            /* Lets start with just changing the div -> effect changes all div tags */
            div{
                color: green; /* This chnages the fontcolor of all divs to green*/
            }

            .classNameHere{
                color: purple; /* This changes the fontcolor of all tags with the called classname to pruple */
            }

            #idHere{
                color: red; /* This changes the font color of tags with the Id to red*/
            }
